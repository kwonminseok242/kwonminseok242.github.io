반도체 OHT 이송장치 센서 기반 탄화 예측 시스템

OHT 장치 센서 데이터를 분석해 고장 전조 신호(탄화)를 조기에 감지하고, 운영/모니터링 관점까지 고려한 예측 시스템을 목표로 했습니다.

#Python #PyTorch #TimeSeries #PredictiveMaintenance #Monitoring

Project Overview

반도체 생산라인에서 웨이퍼를 이송하는 OHT 장비는 공정 안정성과 직결됩니다. 이 프로젝트는 OHT 장치의 센서 데이터를 기반으로 고장 전조 현상인 탄화를 예측하고, 현업에서 활용 가능한 형태(모니터링/알림/리포트)에 가까운 결과를 목표로 했습니다.

핵심 포인트
(1) 센서 시계열 전처리/피처 설계 → (2) 예측 모델링 → (3) 결과 해석/운영 적용(모니터링 관점)까지 일관된 흐름으로 구성

Core Tasks

  • 데이터 수집/정합 : 장비 센서 로그를 분석 단위로 맞추고 결측/노이즈를 처리
  • 전처리/피처링 : 시계열 특성을 반영한 윈도우링, 통계/추세 기반 피처 생성
  • 모델링 전략 : 탄화 발생 전후 패턴 차이를 포착할 수 있도록 예측/분류 접근을 설계
  • 성능 평가 : 운영 관점에서 중요한 지표(조기 탐지/오탐 최소화)를 중심으로 평가 프레임 구성
  • 결과 커뮤니케이션 : 원인 추정/패턴 설명이 가능하도록 주요 신호를 요약하고 리포트 형태로 정리

Results & Learnings

본 프로젝트를 통해 “모델 정확도”뿐 아니라, 실제 현장에서 쓰이려면 데이터 품질/지표 정의/운영 프로세스(모니터링·알림)까지 함께 설계되어야 한다는 점을 배웠습니다.

추가로 넣으면 더 강력해지는 항목
여기 섹션에 너가 가진 실제 수치(예: 조기탐지 리드타임, 정밀도/재현율, 오탐률 감소 등) 2~3개만 넣으면 포트폴리오 완성도가 확 올라가.
